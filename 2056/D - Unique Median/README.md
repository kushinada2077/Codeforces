<div class="header"><div class="title">D. Calculate the number of good subarrays$$$^{\text{∗}}$$$ in $$$a$$$.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$An array $$$x$$$ is a subarray of an array $$$y$$$ if $$$x$$$ can be obtained from $$$y$$$ by the deletion of several (possibly, zero or all) elements from the beginning and several (possibly, zero or all) elements from the end. </p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases.
